Auto-updating fixture calendar — subscribe once, stay current.

Subscribe in Google Calendar

  1. Open Google Calendar on desktop
  2. Click + next to "Other calendars" → From URL
  3. Paste the URL below and click Add calendar
https://calendar.increlytics.com/lfc.ics

Google Calendar refreshes subscribed feeds roughly every 12-24 hours.

Subscribe in Apple Calendar

  1. On your Mac: File → New Calendar Subscription
  2. On iPhone/iPad: Settings → Calendar → Accounts → Add Account → Other → Add Subscribed Calendar
  3. Paste the same URL above

How it works

A scheduled job runs daily, fetches the latest fixture data from ESPN, and publishes an updated .ics file. Your calendar app picks up changes automatically — new fixtures, rescheduled kick-offs, and results.
